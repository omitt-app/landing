# Security

## Supported version

Security fixes apply to the current deployment at [omitt.app](https://omitt.app) and the latest revision of `main`.

## Report an issue privately

Please email [hello@omitt.app](mailto:hello@omitt.app) with:

- A clear description of the issue and its impact
- The affected page or workflow
- Reproduction steps or a minimal proof of concept
- Your preferred contact details for follow-up

Do not open a public issue for a vulnerability. Do not include real customer data, document contents, passwords, private keys, access tokens, or other sensitive material in a report.

We will acknowledge useful reports, investigate them carefully, and coordinate remediation before public discussion when disclosure is appropriate. omitt does not currently operate a paid vulnerability-reward program.

## Privacy boundary

The public website does not process documents. The macOS app performs detection and redaction on-device. Waitlist submission is explicit. Choosing Send feedback opens a draft in the user's mail app, and nothing is sent unless the user sends that email, as described in the [privacy notice](https://omitt.app/privacy).
